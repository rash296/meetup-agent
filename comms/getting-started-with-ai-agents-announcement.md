# Getting Started with AI Agents

**You know Python. Now teach it to think for itself.**

If you've been curious about AI agents — the kind that reason through problems, call tools, and chain together multi-step workflows — this is your starting point. No prior AI experience needed, just bring your Python skills and a laptop.

## What's Happening

**Getting Started with AI Agents** is a hands-on workshop where you'll build a working AI agent from scratch using Claude. In 2.5 hours, you'll go from "what even is an agent?" to having one that uses tools, manages memory, and orchestrates its own tasks. You'll learn the core patterns behind tool use, autonomous reasoning, and agent workflows — then actually build them.

This isn't a lecture. You'll be writing code the entire time.
