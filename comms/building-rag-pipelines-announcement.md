# Building RAG Pipelines: From Embeddings to Production

**You know how to call an LLM API. But can you build a RAG pipeline that actually works in production?** Most tutorials stop at "just use embeddings" — we're going to build the whole thing, from document chunking to vector storage to retrieval strategies that don't hallucinate.

This is a 3-hour hands-on workshop. You'll build a retrieval-augmented generation pipeline from scratch, covering every layer: how to chunk documents intelligently, store and query vector embeddings, design retrieval strategies that return the right context, and wire it all into an LLM for grounded, accurate answers.

## What You'll Build

Over three sessions, you'll go from raw documents to a working RAG pipeline:

1. **Document Chunking & Embeddings** — how to split documents so retrieval actually finds what you need, and which embedding strategies work for different content types
2. **Vector Storage & Retrieval** — setting up a vector database, writing queries that return relevant results, and understanding when knowledge graphs complement vector search
3. **LLM Integration & Production Patterns** — connecting retrieval to generation, handling edge cases, and making it production-ready
